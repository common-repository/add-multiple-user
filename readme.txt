== Description ==

= Add multiple users in bulk =
* When registering one parent user, register multiple child users.
* Multiple users can be registered at once with the same mail address.
* Can specify the roles of the parent user and child user.
* Deleting the parent user also deletes the child user.

= Cooperation with other plugins =
* Work with [User Mail Only Register](https://wordpress.org/plugins/user-mail-only-register/).
* Work with [Software License Manager Client](https://wordpress.org/plugins/software-license-manager-client/).
* Work with [Woocommerce](https://wordpress.org/plugins/woocommerce/).
* The additional child users number can be changed from the user by charging.

== Installation ==

1. Upload `add-multiple-user` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
